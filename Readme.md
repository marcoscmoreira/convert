### Objetivo da aplicação
- Converter valor da moeda para reais(BRL) baseado na
quantidade inserida pelo usuário.

### Moedas convertidas
- BRL (Real Brasileiro)
- USD (Dólar Americano)
- € (Euro)
- £ (Libra Esterlina)
